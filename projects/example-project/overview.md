# Example Project

## Кратко

Замените этот пример на свой проект, бизнес-направление, продукт или исследование.

## Что хранить здесь

- цель проекта;
- гипотезы;
- задачи;
- решения;
- ссылки на отдельные репозитории;
- handoff и следующий шаг.

## Когда выносить в отдельный репозиторий

Если появляются исходный код, зависимости, тесты, сборка, деплой или отдельные права доступа, создайте отдельный репозиторий из `templates/project-repo/`.
