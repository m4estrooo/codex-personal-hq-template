# Общие правила работы

## Язык

- По умолчанию все ответы и документы ведутся на русском.
- Английские термины допустимы, если они общеприняты, но при первом использовании нужно дать русский смысл.

## Факты

- Не использовать память модели как источник актуальных фактов.
- Для цен, объявлений, тарифов, моделей, видеокарт, сервисов, правил площадок и новостей нужна проверка.
- Если проверка невозможна, это нужно явно написать.

## Файлы

- Не создавать файлы без понятного назначения.
- Не плодить черновики в корне проекта.
- Временные материалы хранить в папках, исключенных из Git.
- Завершенные задачи переносить в архив.

## Безопасность

- Не хранить секреты в репозитории.
- Не публиковать приватную информацию без явного решения владельца репозитория.
- При сомнительных действиях выбирать безопасный вариант и фиксировать риск.
