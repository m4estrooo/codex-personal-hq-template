# Протокол исследований

## Когда обязателен

Протокол обязателен для всего, что быстро меняется:

- цены;
- объявления;
- тарифы;
- наличие;
- видеокарты и GPU-аренда;
- AI-модели;
- правила площадок;
- новости;
- версии библиотек;
- рекомендации сервисов.

## Формат источника

```md
## Источник

- Название:
- Ссылка:
- Проверено: YYYY-MM-DD HH:mm Europe/Minsk
- Метод проверки: браузер / официальный сайт / API / поиск / скриншот
- Статус: актуально / неактуально / не удалось проверить
- Риск устаревания: низкий / средний / высокий
- Краткий вывод:
```

## Правила

- Не писать "актуально", если источник не проверен в текущей задаче.
- Для объявлений и маркетплейсов фиксировать, что статус был верен только на момент проверки.
- По возможности предпочитать официальные источники, первоисточники и живой браузер.
- Если источник не открывается, фиксировать это как результат проверки.
