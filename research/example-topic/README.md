# Example Research

## Цель

Замените этот пример на свое исследование.

## Главные вопросы

- Что нужно проверить?
- Какие источники считаются надежными?
- Какие факты могут быстро устареть?
- Какой следующий практический шаг?

## Правило

Все выводы по динамическим фактам должны иметь дату, источник, метод проверки и риск устаревания.
