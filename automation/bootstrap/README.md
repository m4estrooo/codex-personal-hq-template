# Bootstrap

Здесь будут инструкции и скрипты для настройки нового устройства.

## Минимальная цель

- Проверить Git.
- Проверить доступ к GitHub.
- Проверить Codex.
- Проверить глобальный `AGENTS.md`.
- При необходимости скопировать `templates/codex-global/AGENTS.md` в глобальную папку Codex.
- Склонировать или обновить репозиторий штаба.
- Проверить, что новый чат видит инструкции.
