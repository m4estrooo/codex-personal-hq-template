# PROJECT_NAME

Краткое описание проекта: что он делает, для кого, какую проблему решает.

## Быстрый старт для Codex

1. Прочитай `AGENTS.md`.
2. Прочитай `handoff/current.md`.
3. Открой активную задачу из `tasks/active/`.
4. Создай видимый план Codex. Если план-трекер недоступен, веди чеклист в активной задаче.

## Структура

- `AGENTS.md` - правила Codex для этого проекта.
- `docs/` - архитектура, решения, заметки и справка.
- `tasks/` - входящие, активные, отложенные и архивные задачи.
- `handoff/` - передача контекста между чатами, устройствами и аккаунтами.
- `src/` - исходный код, если проект кодовый.
- `tests/` - тесты, если они есть.
